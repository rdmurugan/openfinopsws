Run Your Own Demo

Get the demo running on your local machine in 3 simple steps

1

Install

Install FinOpsMetrics via pip

pip install finopsmetrics
2

Start Server

Launch the Web UI server

finopsmetrics server
3

Open Browser

Access the dashboards

http://localhost:8080

What You'll Experience

Real-time Updates

WebSocket-powered live updates every 5 seconds. Watch your dashboards update in real-time without page refreshes.

Interactive Charts

Beautiful Chart.js visualizations with tooltips, zoom, and animations. Explore your data with intuitive interactions.

Modern Design

Glassmorphism UI with dark theme, gradients, and smooth animations. A beautiful interface for FinOps intelligence.

Responsive Layout

Fully responsive design that works perfectly on desktop, tablet, and mobile devices. Monitor costs anywhere.

AI Recommendations

Smart cost optimization suggestions powered by AI. Get actionable recommendations with estimated savings.

Role-based Views

Different dashboards for CFO, COO, and Infrastructure teams. Each role gets the metrics that matter most.

System Requirements

  • Python: 3.8 or higher
  • Browser: Chrome, Firefox, Safari, or Edge (latest versions)
  • Network: localhost access or network connection for remote access
  • Resources: Minimal CPU and memory footprint
