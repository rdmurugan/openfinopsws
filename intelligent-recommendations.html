Overview

Quick Start

1. Install OpenFinOps

pip install openfinops[all] # Or with specific LLM provider pip install openfinops openai # For OpenAI pip install openfinops anthropic # For Anthropic

2. Configure LLM (Optional)

# Environment variables export OPENAI_API_KEY="sk-..." export OPENAI_MODEL="gpt-4-turbo-preview" # Or use config file (llm_config.json) { "provider": "openai", "model_name": "gpt-4-turbo-preview", "api_key": "sk-...", "temperature": 0.3, "max_tokens": 1000 }

3. Get Recommendations

from openfinops.observability.intelligent_recommendations import get_recommendations # Your current infrastructure metrics current_metrics = { 'instance_count': 4, 'avg_cpu_utilization': 65, 'avg_gpu_utilization': 45, 'gpu_count': 1, 'cost_per_instance_hour': 3.06, 'workload_type': 'inference' } # Get intelligent recommendations recommendations = get_recommendations( current_metrics=current_metrics, workload_type='inference', cloud_provider='aws' ) # Display recommendations for rec in recommendations: print(f"\n{rec.title}") print(f"Priority: {rec.priority}") print(f"Impact: ${rec.impact['cost_monthly_usd']}/month") print(f"Description: {rec.description}")

4. Use in Dashboard

from openfinops.dashboard import COODashboard dashboard = COODashboard() # Get recommendations directly in dashboard recommendations = dashboard.get_intelligent_recommendations( current_metrics=current_metrics ) # Display in dashboard print(f"Total Recommendations: {recommendations['summary']['total_recommendations']}") print(f"Potential Savings: ${recommendations['summary']['potential_monthly_savings']:.2f}/month")

Advanced Configuration

Custom LLM Configuration
