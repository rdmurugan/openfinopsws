v0.2.0 Latest
January 15, 2025
New Features
  • Intelligent LLM-Powered Recommendations - Added AI-powered hardware and scaling recommendations using OpenAI, Anthropic, Azure OpenAI, or Ollama
  • Hardware Optimization Engine - Automatic GPU selection and right-sizing recommendations (A100, H100, T4, L4, V100)
  • Scaling Strategies - Intelligent auto-scaling, predictive scaling, and spot instance recommendations
  • Multi-LLM Support - Configurable LLM providers with automatic fallback to rule-based recommendations
  • Data Platform Monitoring - Added support for Databricks and Snowflake cost tracking
  • SaaS Services Tracking - Monitor costs for MongoDB Atlas, Redis Cloud, GitHub Actions, and more
Improvements
  • Enhanced real-time Web UI with WebSocket updates for live cost tracking
  • Improved COO Dashboard with integrated recommendations widget
  • Added export capabilities for recommendations (JSON, Markdown, HTML)
  • Performance optimizations for large-scale deployments
Documentation
  • Comprehensive guide for intelligent recommendations setup
  • LLM configuration examples for all supported providers
  • Updated API documentation with new recommendation endpoints
